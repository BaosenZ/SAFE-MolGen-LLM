# SAFE-fSepaMolGen Website Guide

The website for this work refers to <a href='www.safe.lanl.gov/fsepamolgen/doc' target='_blank'>www.safe.lanl.gov/fsepamolgen/doc</a>. The video guide for the website is <a href='https://youtu.be/9Ut_G-mYzC0' target='_blank'>https://youtu.be/9Ut_G-mYzC0</a>.  

## How to Use Our Service

1. Log in to the SAFE website at https://safe.lanl.gov/login. If you don't have an account yet, please register first at https://safe.lanl.gov/signup.
2. Go to the SAFE-fSepaMolGen website: https://safe.lanl.gov/fsepamolgen.
3. On the website, choose your preferred LLM and provide the corresponding API key.
    - If you select a GPT model, use an API key from the OpenAI platform.
    - If you choose a Llama model, the key should come from your NERSC account (know more [NERSC](https://www.nersc.gov/)).  
    - We provide [video guides](https://youtu.be/9Ut_G-mYzC0) for both options on the website. Note that we do not store your API key.
4. You can enter your parameters in the input fields provided to run the workflow.
5. Click the "Generate New Extractants" button to start the generation process. Optionally:
    - Click "Preview Experimental Evaluation Table for Prompt" to view the evaluation data and associated DOIs.
    - Click "Preview LLM's Prompt" to see the LLM prompt that will be sent to the model.
6. Please wait for ~10 minutes as it will generate and evaluate the new extractants.
7. Once completed, you will receive your results in your registered email. You can download your results (including New Extractants and corresponding experimental separation conditions) and parameters from your email.
